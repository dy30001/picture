# 前端业务模块边界

后续前端按下面 5 个模块拆分：

1. `studio`
   - 拍摄定制、样片集、样片状态管理、三视图、选模板、生成成片。
   - 当前样片管理 MVP 先在 `public/app.js` 中落地，迁移时再拆到 `studio/sample-management`。
2. `image-generation`
   - 模板库、智能生图、换背景、换装、高清增强、局部修复。
3. `works`
   - 我的作品、全部作品、已删除、预览、恢复、清除。
4. `auth`
   - 注册开通、注册、登录、邮箱验证码。
5. `credits`
   - 我的积分、买积分；充值记录合并在“买积分”里，积分明细合并在“我的积分”里。

迁移时每次只迁一个模块，并配套测试和视觉检查。
