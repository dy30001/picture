# 宣传理念与对外表达规范

日期：2026-05-05
模式：`doc`
工作区：`/Users/dy3000/code/pic`

## 0. 结论

墨境对外不讲“我有很多模板、我能调模型”，只讲客户能得到的生活结果：

```text
让向往的生活，先在照片里发生。
```

这句话是当前统一品牌主张。首页、宣传页、分享卡、样片页和对外说明都围绕它展开。

## 1. 宣传定位

墨境是“个人AI摄影棚”，不是普通图片生成工具。

对外表达应让客户形成三个判断：

1. 这是一次可交付的影像定制服务。
2. 客户只需要提交照片和影像方向。
3. 最终得到的是能保存、下载、分享和交付的正式成片。

不把客户带进这些问题：

- 模型怎么选
- prompt 怎么调
- 接口怎么测
- 后台怎么制作
- 过程图怎么筛

## 2. 核心主张

固定主张：

```text
让向往的生活，先在照片里发生
```

含义：

- `向往的生活`：客户想补上的关系、纪念、形象和场景。
- `先在照片里发生`：让没能留下、来不及同框或不够满意的内容先成为可见作品。

这句主张优先用于：

- 首页首屏标题
- 分享卡主标题
- 宣传海报标题
- 对外介绍的第一句话
- 客户转介绍话术

## 3. 对外只讲四类价值

### 3.1 修复旧照

客户感知：

- 旧照片变清楚。
- 破损、模糊、低清的照片得到修复。
- 重要记忆可以重新保存。

推荐表达：

- `旧照修复，让老照片重新清楚起来`
- `模糊的记忆，变成能保存的作品`
- `把珍贵旧照修成清晰版本`

### 3.2 亲人同框

客户感知：

- 想念的人可以在照片里重新同框。
- 没有合影的遗憾可以补上。
- 家庭纪念照更完整。

推荐表达：

- `没有留下的合影，现在补成一张`
- `把想念的人，放回同一张照片里`
- `让家人的重要时刻重新同框`

### 3.3 爱人合影

客户感知：

- 没有正式婚纱照、旅行纪念照的人可以补成一组作品。
- 爱情和纪念日有更好看的照片。
- 不需要预约影棚和奔波选景。

推荐表达：

- `错过的婚纱照，现在补上`
- `想去的地方，先变成旅行大片`
- `把纪念日变成正式作品`

### 3.4 正式成片

客户感知：

- 不是随便玩一张图，而是有正式结果。
- 有成片列表、下载、反馈和交付记录。
- 作品可以保存和复用。

推荐表达：

- `正式成片交付`
- `可下载、可保存、可复看的作品`
- `制作完成后，直接查看成片`

## 4. 页面表达规则

### 4.1 首页

首页只讲愿景和入口：

- 主标题：品牌主张。
- 副标题：客户能得到什么。
- 主动作：进入影像定制或图片创作。
- 证据：样片、成片、服务类型。

首页不讲：

- 分步骤教程
- 技术连接
- 后台制作过程
- 长 prompt
- 内部端口
- 调试状态

### 4.2 样片页

样片页只承担“看效果”和“选方向”。

允许出现：

- 场景分类
- 样片组
- 适合人群
- 风格标签
- 放大预览

不出现：

- 生成按钮
- 后台状态
- 过程稿
- 模板 ID
- 失败图
- 内部审核词

### 4.3 提交页

提交页只讲客户要交什么：

- 影像组合
- 每个人照片
- 风格方向
- 客户备注

主按钮统一为：

```text
提交定制需求
```

### 4.4 状态页

状态页只讲客户当前能理解的状态：

- `已提交`
- `资料需补充`
- `制作中`
- `已完成`

不把后台状态原样透传到客户前台。

### 4.5 成片页

成片页只讲正式交付：

- 成片数量
- 成片列表
- 下载入口
- 反馈入口
- 通知记录

过程稿、失败图、工作人员备注和调试日志不进入成片页。

## 5. 禁用宣传方向

不要把宣传写成这些方向：

| 禁用方向 | 问题 |
| --- | --- |
| `2300+ 模板随便选` | 客户会感觉仍在操作工具 |
| `GPT Image 2 参数很强` | 技术词抢走服务价值 |
| `先上传、再选择、再等待` | 像教程，不像宣传 |
| `后台审核三视图` | 暴露内部生产动作 |
| `连接设置、测试连接` | 客户前台不应承担技术配置 |
| `失败重跑、模型试错` | 降低客户信任 |

技术能力可以写在 README 的内部调试区，不进入客户宣传面。

## 6. 推荐短句库

主张类：

- `让向往的生活，先在照片里发生`
- `把想要的生活，变成看得见的作品`
- `把遗憾补成一张正式照片`

服务类：

- `旧照修复`
- `亲人同框`
- `爱人合影`
- `婚纱旅拍`
- `家庭纪念`
- `正式成片`

动作类：

- `看效果`
- `提交资料`
- `制作状态`
- `查看成片`
- `提交定制需求`
- `买积分`

质量类：

- `像本人`
- `皮肤自然`
- `关系自然`
- `构图高级`
- `可下载保存`
- `可复看交付`

## 8. 完成标准

后续新增对外文案时，必须满足：

1. 先讲结果，不讲技术。
2. 先讲生活愿景，不讲流程教程。
3. 客户能一眼知道能得到什么。
4. 不出现后台制作动作。
5. 不出现模型参数、接口测试和内部端口。
6. 语气紧凑，不堆长段解释。
