# 客户语言口径规范

日期： 2026-05-03
模式： `doc`
工作区： `/Users/dy3000/code/pic`

## 0. 目的

这份规范只管一件事：客户界面和对外说明里，只保留客户能直接理解的词，不再混用内部逻辑词。

## 1. 总原则

- 只说动作、结果和状态。
- 只保留客户能扫一眼就懂的词。
- 每个功能独立展示，不把多个功能混成一页。
- 客户界面不解释系统，不讲内部字段名。

## 2. 不出现在客户界面的词

- `账户权益`
- `权益总览`
- `余额`
- `三视图`
- `数字底片`
- `加料项`
- `继续`
- `第 N 步`
- `生成成片`
- `主提示词`
- `加料`

## 3. 推荐写法

| 内部词 | 客户写法 |
| --- | --- |
| `账户权益` | `我的` / `我的积分` |
| `权益总览` | `我的积分` |
| `余额` | `可用积分` |
| `三视图` | `身份确认` / `确认像本人` |
| `数字底片` | `参考照` |
| `加料项` | `可选功能` |
| `继续` | `下一步` / 具体动作 |
| `第 N 步` | `步骤 N` / `当前步骤` |
| `生成成片` | `开拍成片` / `生成` |
| `主提示词` | `画面描述` / `想拍什么` |
| `加料` | `可选功能` / `附加效果` |

## 4. 现在统一口径

- 一级菜单：`拍摄定制` / `图片创作` / `我的`
- 拍摄定制：`样片集`、`身份确认`、`样片方向`、`成片交付`
- 我的：`我的作品`、`注册`、`登录`、`我的积分`、`充值中心`、`充值订单`、`积分流水`
- 积分页只说 `我的积分`、`可用积分`、`购买积分`、`出图成功才扣积分`
- 可选编辑只说 `可选功能`
- 智能生图主区优先说 `上传照片`、`画面描述`、`开拍`
- `提示词怎么写`、`模板怎么借`、`写法参考` 默认归到侧边辅助说明，不写成主区大段解释
- `图片创作` 首页主区统一写 `趣味生图`、`套图写真`
- `模板库` 只作为现阶段路由名、次级入口或筛选口径，不再当客户首页主标题
